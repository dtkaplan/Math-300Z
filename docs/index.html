Take-aways from class 19

Statistical

  1. Measure variation using the “variance.” In R,
Galton %>% summarize(vh = var(height))

  1. Response variable will always be quantitative/numerical in a regression model.

  2. Expect graphics in our section to always have the response variable on the vertical axis.

Administrative

  1. DO THE DAILY WORKSHEET! You’ll fall behind if you don’t.

  2. Bookmark this class site in your browser. That way you don’t have to detour through Teams.

  3. Use the Z-section project on posit.cloud. Bring in the worksheet using

math300::get_lesson_worksheet( LESSON NUMBER )
